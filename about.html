WHAT DO WE DO?

Coding cadets works to spread knowledge of programming to a wide range of students. We teach languages like HTML, CSS, and Javascript along with supplementary resources like Bootstrap, Font Awesome, and Meteor to empower students with the ability to create advanced websites and web applications.

By the end of our first beginning course, students will be able to create responsive static websites following common web design principles and standards. By the end of our intermediate course, students will be able to create interactive web applications that can be deployed online.

WHERE ARE WE?

Currently, we teach at Cypress Library in Cypress, CA. Our classes take place on Tuesdays from 4-5 p.m. and Saturdays 10-12:30 p.m. Students can sign up at the library a few weeks prior to the beginnign of a new session.

It is possible that we may expand to more locations soon.
